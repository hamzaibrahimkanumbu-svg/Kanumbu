JavaScript in HTML

A Comprehensive Tutorial with Interactive Examples

1. Including JavaScript in HTML

JavaScript can be included in HTML documents in three ways:

Inline JavaScript

JavaScript code can be placed directly within HTML elements using event attributes.

<button onclick="alert('Hello World!')">Click Me</button>

Internal JavaScript

JavaScript code can be placed within <script> tags in the HTML document.

<script>
  // This is internal JavaScript
  function greet() {
    alert('Hello from internal JavaScript!');
  }
</script>

External JavaScript

JavaScript code can be placed in a separate file and linked to the HTML document.

<script src="script.js"></script>

Best Practice: For better organization and performance, use external JavaScript files.

2. DOM Manipulation

The Document Object Model (DOM) represents the structure of an HTML document. JavaScript can manipulate the DOM to change content, styles, and structure.

Selecting Elements

// Select by ID
let element = document.getElementById('myId');

// Select by class name
let elements = document.getElementsByClassName('myClass');

// Select by tag name
let elements = document.getElementsByTagName('div');

// Modern selectors (similar to CSS)
let element = document.querySelector('#myId');
let elements = document.querySelectorAll('.myClass');

3. Event Handling

JavaScript can respond to user interactions through events.

Common Events

  • click - When an element is clicked
  • mouseover - When mouse pointer moves over an element
  • mouseout - When mouse pointer moves out of an element
  • keydown - When a key is pressed down
  • load - When the page has finished loading

Event Listeners

// Adding an event listener
document.getElementById('myButton').addEventListener('click', function() {
  console.log('Button clicked!');
});

4. Variables and Data Types

JavaScript variables can hold different types of data.

Variable Declaration

// Using var (older way, function-scoped)
var name = "John";

// Using let (modern, block-scoped)
let age = 25;

// Using const (for constants, cannot be reassigned)
const PI = 3.14159;

Data Types

// String
let message = "Hello World";

// Number
let count = 42;

// Boolean
let isActive = true;

// Array
let colors = ["red", "green", "blue"];

// Object
let person = {
  name: "Alice",
  age: 30,
  isStudent: false
};

5. Functions

Functions are reusable blocks of code that perform a specific task.

Function Declaration

// Function declaration
function greet(name) {
  return "Hello, " + name + "!";
}

// Function expression
const multiply = function(a, b) {
  return a * b;
};

// Arrow function (ES6)
const divide = (a, b) => a / b;

6. Conditional Statements

Conditional statements allow your code to make decisions.

If-Else Statement

let temperature = 25;

if (temperature > 30) {
  console.log("It's hot outside!");
} else if (temperature > 20) {
  console.log("The weather is pleasant.");
} else {
  console.log("It's cold outside!");
}

Switch Statement

let day = "Monday";

switch(day) {
  case "Monday":
    console.log("Start of the work week");
    break;
  case "Friday":
    console.log("Weekend is almost here!");
    break;
  default:
    console.log("It's a regular day");
}

7. Loops

Loops allow you to execute a block of code multiple times.

For Loop

// Print numbers from 1 to 5
for (let i = 1; i <= 5; i++) {
  console.log(i);
}

While Loop

// Print numbers from 1 to 5
let i = 1;
while (i <= 5) {
  console.log(i);
  i++;
}
